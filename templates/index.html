Ejercicio 1

Ejercicio 1 :

Enunciado del Ejercicio:

  • Utilizando los comandos CREATE DATABASE, CREATE TABLE e INSERT INTO, crea una base de datos llamada "Ferretería" que contenga dos tablas, una llamada "Inventario" y otra llamada "Proveedores".
  • Cada tabla debe contener al menos con cinco filas.
  • Para la tabla "Inventario" crea un campo cuyos valores comiencen en 1 y vayan incrementando de uno en uno automáticamente y utiliza este campo como llave primaria. Agrega además los campos "Descripción", "Inventario_Minimo" y "Proveedor".
  • La tabla "Proveedores" deberá contener los campos "Proveedor" (el cual contendrá la misma información que el campo "Proveedor" de la tabla "Inventario"), "Teléfono" y "Tiempo de Entrega", que será la cantidad de días hábiles que el proveedor tarda en entregar el material a la Ferretería.

                      CREATE DATABASE FERRETERIA;
                      
                      USE FERRETERIA;
                      
                      CREATE TABLE Inventario
                          (
                            ID INT IDENTITY(1,1),  
                            Descripcion VARCHAR(255),
                            Inventario_Minimo INT,
                            Proveedor VARCHAR(255),
                            PRIMARY KEY(ID)
                          );
                      
                      CREATE TABLE Proveedores
                          (
                            Proveedores varchar(255),
                            telefono varchar(30),
                            Tiempo_de_Entrega int
                          );
                      
                      
                      INSERT INTO Inventario
                          (Descripcion,Inventario_Minimo,Proveedor )
                            values				
                                ('Producto 1', 10, 'Proveedor 1'),
                                ('Producto 2', 20, 'Proveedor 2'),
                                ('Producto 3', 30, 'Proveedor 3'),
                                ('Producto 4', 40, 'Proveedor 4'),
                                ('Producto 5', 50, 'Proveedor 5')
                      
                      INSERT INTO Proveedores

Ejercicio 3:

Enunciado del Ejercicio:

Para llevar a cabo esta tarea, vamos a utilizar la tabla MaratonNY que se encuentra en la base de datos Maraton. Teniendo esto en cuenta realiza lo siguiente:

  • Obtén el tiempo que le tomó llegar a la meta a cada una de las mujeres provenientes de NY. Muestra las columnas gender, age, home y time en los resultados y ordénalos de forma ascendente en base a los valores de la columna time.
                
                      SELECT 
                          D.gender,
                          D.age, 
                          D.home,
                          D.[time] 
                      FROM
                          DBO.MaratonNY AS D
                      WHERE 
                          D.gender = 'Female' AND
                          D.home='NY'
                      ORDER BY D.time ASC

  • Crea una lista con los hombres menores de 30 años y de las mujeres que tengan entre 30 y 40 años. Incluye las columnas gender, age y home. Ordena la lista en base a la edad.
              
                    SELECT

  • Realiza una subconsulta escalonada para obtener una lista de los corredores mayores de 25 años que sean mas veloces que los corredores menores de 25 años. Indica cual es el tiempo mínimo que hicieron éstos últimos y después incluye las columnas gender, age y time en tus resultados.
              
                    SELECT 
                        D.gender,
                        D.age,
                        D.[time]
                    FROM 
                        DBO.MaratonNY AS D
                    WHERE 
                        age > 25 AND 
                        time < (SELECT MIN(time) FROM DBO.MaratonNY WHERE age < 25);

  • Realiza una subconsulta de lista para conocer cuales hombres menores de 40 años fueron mas lentos que todas las mujeres menores de 40 años. Ordénalos de forma descendente en base a la columna age.
              
                      SELECT 
                          D.gender, 
                          D.age, 
                          D.[time]
                      FROM
                          DBO.MaratonNY AS D
                      WHERE 
                          gender = 'Male' AND
                          age < 40 AND time > ALL
                          (SELECT time FROM DBO.MaratonNY WHERE gender = 'Female' AND age < 40)
                      ORDER BY 
                      age DESC;

                    CREATE VIEW MI_VISTAS AS
                    SELECT
                        C.FirstName,C.LastName,C.CompanyName,C.EmailAddress,
                        OC.OrderQty, OC.PurchaseOrderNumber,
                        P.[Name], P.ProductNumber, P.ProductID
                    FROM
                      dbo.Clientes AS C
                    INNER JOIN 
                      dbo.OrdenesdeCompra as OC
                    ON
                      C.CustomerID = OC.CustomerID
                    INNER JOIN
                      DBO.Productos AS P
                    ON 
                      P.ProductID = OC.ProductID
                        
                    SELECT * FROM MI_VISTAS
