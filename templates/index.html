Ejercicio 1

Ejercicio 1 :

Enunciado del Ejercicio:

  • Utilizando los comandos CREATE DATABASE, CREATE TABLE e INSERT INTO, crea una base de datos llamada "Ferretería" que contenga dos tablas, una llamada "Inventario" y otra llamada "Proveedores".
  • Cada tabla debe contener al menos con cinco filas.
  • Para la tabla "Inventario" crea un campo cuyos valores comiencen en 1 y vayan incrementando de uno en uno automáticamente y utiliza este campo como llave primaria. Agrega además los campos "Descripción", "Inventario_Minimo" y "Proveedor".
  • La tabla "Proveedores" deberá contener los campos "Proveedor" (el cual contendrá la misma información que el campo "Proveedor" de la tabla "Inventario"), "Teléfono" y "Tiempo de Entrega", que será la cantidad de días hábiles que el proveedor tarda en entregar el material a la Ferretería.

              CREATE DATABASE FERRETERIA;
              
              USE FERRETERIA;
              
              CREATE TABLE Inventario
                  (
                    ID INT IDENTITY(1,1),  
                    Descripcion VARCHAR(255),
                    Inventario_Minimo INT,
                    Proveedor VARCHAR(255),
                    PRIMARY KEY(ID)
                  );
              
              CREATE TABLE Proveedores
                  (
                    Proveedores varchar(255),
                    telefono varchar(30),
                    Tiempo_de_Entrega int
                  );
              
              
              INSERT INTO Inventario
                  (Descripcion,Inventario_Minimo,Proveedor )
                    values				
                        ('Producto 1', 10, 'Proveedor 1'),
                        ('Producto 2', 20, 'Proveedor 2'),
                        ('Producto 3', 30, 'Proveedor 3'),
                        ('Producto 4', 40, 'Proveedor 4'),
                        ('Producto 5', 50, 'Proveedor 5')
              
              INSERT INTO Proveedores 
                   (Proveedores,telefono,Tiempo_de_Entrega )
                       values
                          ('Proveedor 1', '1234567890', 2),
                          ('Proveedor 2', '0987654321', 3),
                          ('Proveedor 3', '1122334455', 4),
                          ('Proveedor 4', '5566778899', 5),
                          ('Proveedor 5', '2233445566', 6);

Ejercicio 2

Ejercicio 2 :

Enunciado del Ejercicio:

  • Carga el archivo PODateTime.csv que viene en la carpeta recursos del curso siguiendo las instrucciones que ya vimos anteriormente. Una vez que lo hayas cargado, genera una consulta en la que extraigas el año en el que fueron colocadas las órdenes que comienzan con PO5. Incluye dichas órdenes de compra en tus resultados.
  • Crea una base de datos llamada Maratón y carga dentro de ella el archivo Maraton NY.csv llamando a la tabla MaratonNY, y obtén una lista de los competidores de sexo masculino, que tengan 60 o más años y que hayan llegado a la meta en menos de 200 minutos.
  • Utilizando la misma tabla MaratonNY que creaste en el punto número 2, crea una lista que contenga la cantidad de mujeres provenientes de cada pais que participaron el maratón y ordénala según las cantidades en forma descendente.
  • Realiza la misma consulta que el punto 3, pero obteniendo solamente la información de los paises cuya cantidad de mujeres participantes sea mayor a 10.
  • Obtén la cantidad de participantes por pais, la edad mínima, la edad máxima y la edad promedio de los mismos por cada uno de los países. Ordena la información de manera Descendente en base a la edad máxima por pais para descubrir la procedencia de los participantes de mayor edad en el Maratón de Nueva York.
